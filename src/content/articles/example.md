# Welcome to My Blog

This is an example article. You can create new articles by adding Markdown files to the `src/content/articles/` directory.

## Features

- Markdown support
- Automatic date sorting
- Tag support
- Clean, readable design

## Getting Started

To create a new article:

1. Create a new `.md` file in `src/content/articles/`
2. Add frontmatter with title, date, and optional description/tags
3. Write your content in Markdown
4. The article will automatically appear in the articles list!
